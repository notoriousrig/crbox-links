Add to crbox-links

Drag this button into your bookmarks bar. Click it on any page to save that page as a bookmark.

+ Save to crbox-links

The bookmarklet opens crbox-links in a new tab with the page URL and title pre-filled, ready to pick a category and tags.

If your browser blocks the drag, right-click the button → Bookmark this link.

How it works

The JS payload:

javascript:(function(){var u=encodeURIComponent(location.href),t=encodeURIComponent(document.title);window.open('https://links.crbox.ca/?share=1&url='+u+'&title='+t,'_blank','noopener');})();

The app reads ?share=1&url=…&title=… on load and opens the add-bookmark modal pre-filled.
